## 🚀 طرق النشر السريعة

### 1. النشر التلقائي (الأسهل)
```bash
npm run deploy
npm start
```

### 2. النشر بـ Docker
```bash
docker-compose up -d
```

### 3. النشر بـ PM2
```bash
npm install -g pm2
pm2 start server.js --name "tamsik"
```

### 4. النشر التقليدي
```bash
npm install
npm start
```
